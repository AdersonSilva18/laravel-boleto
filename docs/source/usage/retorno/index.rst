Retorno
=======

There are 2 types of ``Retornos``: 240 positions and 400 positions.

**Options available by bank:**

=========================  ====  ====
Banco                      240   400
=========================  ====  ====
Bancoob                    yes   yes
Banrisul                   yes*  yes*
Banco do Brasil            yes   yes*
Banco do Nordeste          no    no
Bradesco                   yes   yes*
Caixa Econônica Federal    yes   yes*
HSBC                       no    yes
Itáu                       yes   yes*
Santander                  yes   yes
Sicredi                    yes*  yes*
=========================  ====  ====

.. note::
    *** requires homologation**
